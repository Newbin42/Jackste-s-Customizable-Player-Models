# Jackste's Customizable Player Models
This is where I'm storing every model I make for the Customizable Player Models mod.

# How to Use
## Via Skin
Upload the skin's png file to your Minecraft account.

## Via Model

<details>
<summary>Instructions</summary>
  
1. Download the model (`model.cpmmodel`)
2. Load minecraft instance containing CPM `0.6.25a`
3. Enter the skin editor  
   <img width="221" height="102" alt="image" src="https://github.com/user-attachments/assets/b75950bc-ea37-4374-9cec-45015adb3ead" />   
4. Go to `Edit` -> `Models`  
   <img width="213" height="656" alt="image" src="https://github.com/user-attachments/assets/91980617-c9f2-4dd0-9cd7-8d9f0c64c852" />  
5. `Select` the model you want to load from the menu  
   <img width="692" height="369" alt="image" src="https://github.com/user-attachments/assets/59c7725a-436c-4f8b-a919-ba27ec669405" />  
6. Click `Apply` (Per Instance, model will reset upon relaunch) or `Upload Skin` (Persists between instances)
   <img width="632" height="681" alt="image" src="https://github.com/user-attachments/assets/19033200-6e38-4a1d-9ea3-a609149dad4f" />

</details>

## Via Project
<details>
<summary>Instructions</summary>

1. Download the project (`model.cpmproject`)
2. Load minecraft instance containing CPM `0.6.25a`
3. Enter the skin editor  
   <img width="221" height="102" alt="image" src="https://github.com/user-attachments/assets/b75950bc-ea37-4374-9cec-45015adb3ead" />  
4. Go to `File` -> `Load`  
   <img width="193" height="132" alt="image" src="https://github.com/user-attachments/assets/b470b426-6253-4c86-9aff-d05564ec56c7" />  
5. Browse to the model you want to load and click `Ok`  
   <img width="656" height="488" alt="image" src="https://github.com/user-attachments/assets/93bbe7b2-0f9b-44e5-a1d3-c57fa6ecdda9" />
   
</details>
